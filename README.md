Best Practice Challenge
Aplique los conceptos aprendidos en Clean Code, Refactoring, TDD , Basic Algorithms y Java Fundamentals.

En los commits realizados al repositorio se debe visualizar la aplicación de los conceptos de TDD. (Los casos de prueba asociados deben ser implementados antes de las funcionalidades requeridas).

La calculadora debera aceptar operaciones de + , - , * y / .

La calculadora debera almacenar un log de operaciones (válidas e inválidas).

La calculadora deberá mostrar en pantalla el log de operaciones de acuerdo al tipo de operación ejecutada (PUNTO 5). Este requerimiento deberá ser una opción para elegir dentro de la calculadora.

La solución no requiere una interfaz gráfica.

La solución deberá tener hacer uso correcto de los conceptos aprendidos de GIT. Para este challenge debera hacerse uso del archivo .gitignore.
